B.Sc. Thesis Project: Image Editing System Using Generative Text-to-Image Diffusion Models

In this project, we employ several generative text-to-image diffusion models for a range of image editing tasks, such as object addition/removal, inpainting/outpainting, background replacement, style transfer, and text modification. Our work primarily utilizes models and techniques presented in papers like PowerPaint, InstructPix2Pix, Imagic, and ObjectDrop, testing their capabilities across various editing tasks on custom datasets.

Landscape Image Generation Using Diffusion Models

Implemented the first Diffusion Model proposed in the DDPM paper from scratch for landscape image generation of size 32*32.

Image Captioning Using Vision Transformers

Fine-Tuning YOLOv8 for Enhanced Detection in Crowded Scenes
